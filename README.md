# Proyecto Final - Estructura Web

## Estructura recomendada

- `web/`
  - `css/` → Hojas de estilo (CSS)
  - `js/` → Scripts JavaScript
  - `img/` → Imágenes
  - `index.jsp`, `productos.jsp`, ...
- `src/` → Código fuente Java
- `build/` → Archivos generados

## Recomendaciones
- Usa plantillas JSP para cabecera y pie de página.
- Mantén la lógica Java fuera de los JSP.
- Utiliza frameworks de frontend como Bootstrap para mejorar el diseño.
